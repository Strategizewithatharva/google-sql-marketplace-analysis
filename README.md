
# Marketplace Optimization SQL Project (Google Strategy & Operations Case)

## Overview
This project simulates the analytics work done by Strategy & Operations Analysts at Google.
It includes SQL analyses focused on:
- Marketplace performance
- Funnel analysis
- Operational bottlenecks
- Growth opportunity identification

## Files
- users.csv
- products.csv
- orders.csv
- traffic_logs.csv
- support_tickets.csv
- analysis.sql (main SQL logic)

## How to Use
Run the SQL queries in BigQuery, PostgreSQL, Snowflake, or MySQL.
